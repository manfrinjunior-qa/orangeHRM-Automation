# OrangeHRM Automation - Cypress

Projeto de automação de testes para o sistema OrangeHRM utilizando o framework Cypress.

## Funcionalidades Testadas
* **Login**: Validação de acesso com diferentes perfis.
* **Admin**: Gerenciamento de usuários e validação de elementos de interface.

## Requisitos
* Node.js instalado.
* VS Code ou outro editor de código.

## Instalação e Execução
1. Instale as dependências:
   ```bash
   npm install

    Abra a interface do Cypress:
    Bash

    npx cypress open

    Execute os testes em modo terminal:
    Bash

    npx cypress run
